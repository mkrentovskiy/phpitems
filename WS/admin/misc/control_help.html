Секция управления

В данный раздел вынесены все подсистемы, управление которыми осуществляется специализированными модулями, расширяющие функции CMS. В качестве базовых модулей используются:
  • Резервное копирование - предназначено для сохранение содержимого базы с целью восстановления в случае сбоев.
  • Файловый менеджер - подсистема управления директориями и файлами, расположенными на сервере.
