Система управления сайтом

Для опреатиного внесения информации на сайт посредством вэб-интерфеса используется ситема упрасления содержимым сайта (CMS, Content Managment System). Поскольку сайты, требующие оперативное обновление, имеют сложную систему, доступ к которой разграничен на уровне пользователей, данный механизм разделен на три основных раздела:
  • Система - представляет собой раздел для разработчиков и администраторов системы. Содержит функции для управления пользователями и группами, инструменты измененния объектов предметной области, котроль работы посредством журналирования
  • Информация - раздел, позволяющий оперировать информацией, доступной посетителям на сайте. Предназначен для лиц, обеспечивающий информационную поддержку проекта
  • Контроль - подсистема дополнительных модулей и расширений, позвояющая увеличить функциональность системы
В вверхней части окна отображения страницы расположено поле, содержащее информацию о пользователе и его правах на доступ к каждому из разделов. Слева находюдится меню, позволяющее получить доступ непосредственно к объектам системы.
